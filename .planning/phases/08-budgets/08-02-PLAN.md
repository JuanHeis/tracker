**2. `components/budget-row.tsx`** — Individual budget row with bar + tooltip:

Props:
```typescript
interface BudgetRowProps {
  progress: BudgetProgress;
  onEdit: (category: Category, currentLimit: number) => void;
  onDelete: (category: Category) => void;
}
```

Implementation:
- Layout: Two-row structure per budget
  - **Row 1:** Category color dot (10px circle, `backgroundColor: CATEGORIES[progress.category].color`) + category name (font-medium) + spacer + `FormattedAmount` showing "$spent / $limit" + percentage text + warning icon (if >= 80%) + pencil icon + trash icon
  - **Row 2:** Full-width progress bar below the row

- **Progress bar:** Simple div-based bar:
  ```tsx
  <div className="h-2 w-full rounded-full bg-muted">
    <div
      className="h-full rounded-full transition-all duration-300"
      style={{
        width: `${Math.min(progress.percentage, 100)}%`,
        backgroundColor: barColor,
      }}
    />
  </div>
  ```
  - Bar color logic: `progress.percentage >= 100 ? "rgb(239 68 68)" : progress.percentage >= 80 ? "rgb(245 158 11)" : CATEGORIES[progress.category].color`

- **Warning icon:** Show `AlertTriangle` from lucide-react (h-4 w-4) next to percentage when >= 80%. Color amber for 80-99%, red for 100%+

- **Exceeded text:** When percentage >= 100, show "Excedido en" + `FormattedAmount` of `(spent - limit)` in red text, replacing or below the percentage

- **Pencil icon:** Always visible, `text-muted-foreground hover:text-blue-500` (consistent with Phase 2/3/4 pattern). onClick calls `onEdit(progress.category, progress.limit)`

- **Trash icon:** Always visible, `text-muted-foreground hover:text-red-500`. onClick calls `onDelete(progress.category)`

- **Tooltip on progress bar:** Wrap the bar in `TooltipProvider > Tooltip > TooltipTrigger`. TooltipContent shows the expense breakdown:
  ```tsx
  <div className="space-y-1 text-xs">
    {progress.expenses.map(e => (
      <div key={e.id} className="flex justify-between gap-4">
        <span className="text-muted-foreground">{e.name}</span>
        <span>${e.arsAmount.toLocaleString("es-AR")}</span>
      </div>
    ))}
    {progress.expenses.length === 0 && (
      <span className="text-muted-foreground">Sin gastos en este periodo</span>
    )}
  </div>
  ```
