**2. Step validation functions:**

Create `validateArsStep(data: WizardData): Record<string, string>` — validates arsBalance is a valid number >= 0. Returns errors object (empty = valid).

Create `validateUsdStep(data: WizardData): Record<string, string>` — if usdAmount > 0, globalUsdRate must be > 0. If usdAmount is 0, no validation needed (step was skipped). Returns errors object.

**3. sessionStorage draft persistence:**

```typescript
const DRAFT_KEY = "wizardDraft";

function saveDraft(data: WizardData, currentStep: number): void {
  sessionStorage.setItem(DRAFT_KEY, JSON.stringify({ data, step: currentStep }));
}

function loadDraft(): { data: WizardData; step: number } | null {
  const raw = sessionStorage.getItem(DRAFT_KEY);
  if (!raw) return null;
  try { return JSON.parse(raw); } catch { return null; }
}

function clearDraft(): void {
  sessionStorage.removeItem(DRAFT_KEY);
}
```

**4. commitWizardData function (CRITICAL — atomic write):**

```typescript
export function commitWizardData(data: WizardData): void
```

This function writes ALL localStorage keys in one synchronous block. Use `format` from `date-fns` for date strings.

Build monthlyData object with:
- `salaries: {}` (empty)
- `expenses: []`
- `extraIncomes: []`
- `investments: []`
- `usdPurchases`: If `data.usdAmount > 0`, include one untracked purchase: `{ id: crypto.randomUUID(), date: today (yyyy-MM-dd), arsAmount: 0, usdAmount: data.usdAmount, purchaseRate: 0, origin: "untracked", description: "Saldo inicial USD (wizard)" }`. Otherwise empty array.
- `transfers`: If `data.arsBalance > 0`, include one adjustment: `{ id: crypto.randomUUID(), date: today, type: "adjustment_ars", amount: data.arsBalance, description: "Saldo inicial ARS (wizard)", createdAt: new Date().toISOString() }`. Otherwise empty array.
- `loans: []`
- `salaryOverrides: {}`
- `aguinaldoOverrides: {}`
- `_migrationVersion: 7` (CRITICAL — prevents migration corruption)

Then write all 7 keys:
1. `localStorage.setItem("monthlyData", JSON.stringify(monthlyData))`
2. `localStorage.setItem("globalUsdRate", String(data.globalUsdRate || 0))`
3. `localStorage.setItem("salaryHistory", JSON.stringify({ entries: data.salaryAmount > 0 ? [{ id: crypto.randomUUID(), effectiveDate: currentMonth (yyyy-MM), amount: data.salaryAmount, usdRate: data.globalUsdRate || 0 }] : [] }))`
4. `localStorage.setItem("incomeConfig", JSON.stringify({ employmentType: data.employmentType || "dependiente", payDay: data.payDay || 1 }))`
5. `localStorage.setItem("recurringExpenses", JSON.stringify([]))`
6. `localStorage.setItem("budgetData", JSON.stringify({}))`
7. `localStorage.setItem("lastUsedUsdRate", String(data.globalUsdRate || 0))`

After writing all keys, call `clearDraft()`.
