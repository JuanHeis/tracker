Types:
```typescript
export interface SalaryEntry {
  id: string;
  effectiveDate: string;  // "yyyy-MM" format
  amount: number;
  usdRate: number;
}

export interface SalaryHistory {
  entries: SalaryEntry[];
}

export interface IncomeConfig {
  employmentType: "dependiente" | "independiente";
  payDay: number;  // 1-31
}
```

Pure function `getSalaryForMonth(monthKey: string, salaryHistory: SalaryEntry[], overrides: Record<string, { amount: number; usdRate: number }>)` that:
- Checks overrides first (return with `isOverride: true`)
- Finds most recent entry where `effectiveDate <= monthKey` (string comparison works for yyyy-MM)
- Returns `{ amount, usdRate, isOverride: boolean }`
- Returns `{ amount: 0, usdRate: 0, isOverride: false }` if no entry found

Hook `useSalaryHistory()` that:
- Uses `useLocalStorage("salaryHistory", { entries: [] })` for salary history
- Uses `useLocalStorage("incomeConfig", { employmentType: "dependiente" as const, payDay: 1 })` for config
- Provides `addSalaryEntry(entry)`, `updateSalaryEntry(id, updates)`, `deleteSalaryEntry(id)`
- Provides `setIncomeConfig(config)` 
- Exposes `getSalaryForMonth` bound to current history
- Exports all types
