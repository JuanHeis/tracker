**Plazo Fijo auto-calculation** (read-only):
```typescript
function calculatePFValue(investment: Investment): number {
  if (!investment.tna || !investment.plazoDias || !investment.startDate) {
    return investment.currentValue;
  }
  const totalInvested = investment.movements
    .reduce((sum, m) => m.type === "aporte" ? sum + m.amount : sum - m.amount, 0);
  const elapsedDays = differenceInDays(new Date(), new Date(investment.startDate));
  const daysToUse = Math.min(elapsedDays, investment.plazoDias);
  return totalInvested * (1 + (investment.tna / 100) / 365 * daysToUse);
}
```

**Gain/loss calculation** (computed, not stored):
```typescript
function calculateGainLoss(investment: Investment) {
  const totalInvested = investment.movements.reduce((sum, m) => {
    return m.type === "aporte" ? sum + m.amount : sum - m.amount;
  }, 0);
  const currentVal = investment.type === "Plazo Fijo" ? calculatePFValue(investment) : investment.currentValue;
  const gainLoss = currentVal - totalInvested;
  const percentage = totalInvested > 0 ? (gainLoss / totalInvested) * 100 : 0;
  return { totalInvested, currentVal, gainLoss, percentage };
}
```

**Value outdated check:**
```typescript
function isValueOutdated(investment: Investment): boolean {
  if (investment.type === "Plazo Fijo") return false;
  if (investment.status === "Finalizada") return false;
  return differenceInDays(new Date(), new Date(investment.lastUpdated)) > 7;
}
```

**Render logic:**
- Use local state: `const [editing, setEditing] = useState(false)` and `const [value, setValue] = useState(investment.currentValue)`
- If PF: render read-only `<NumberFlow value={calculatePFValue(investment)} />` — no click handler
- If not PF and not editing: render `<span onClick={() => setEditing(true)} className="cursor-pointer hover:underline">` wrapping NumberFlow
- If editing: render `<Input type="number" autoFocus value={value} onChange/onKeyDown(Enter)/onBlur saves>`
- On save: call `onUpdateValue(investment.id, value)`, setEditing(false)
- Below the value, show gain/loss: green if positive, red if negative. Format: `+$1,234 (+5.2%)` or `-$500 (-2.1%)`. Use NumberFlow for the amounts.
- If `isValueOutdated(investment)`: show `<Badge className="bg-amber-100 text-amber-800 dark:bg-amber-900 dark:text-amber-200">Desactualizado</Badge>` next to the value

**Hydration:** Wrap NumberFlow in a hydration check — only render NumberFlow when hydrated. Show a plain formatted number as fallback during SSR. Use useHydration() hook.

Export `calculatePFValue`, `calculateGainLoss`, and `isValueOutdated` as named exports so InvestmentRow can use them for badges.
