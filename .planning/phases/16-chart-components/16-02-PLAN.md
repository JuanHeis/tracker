**Data transformation function** (inside the file, not exported):
```typescript
function buildInvestmentChartData(
  projections: InvestmentProjection[],
  monthLabels: string[],
  globalUsdRate: number
): { data: Record<string, number | string>[]; types: string[] }
```
Logic:
1. Get unique investment types: `[...new Set(projections.map(p => p.type))]`
2. For each month index (0 to projections[0].projectedValues.length - 1):
   - Create point `{ month: monthLabels[index] || "M" + index }`
   - For each type: sum all projections of that type at month index, converting USD values (`p.currencyType === CurrencyType.USD ? value * globalUsdRate : value`)
   - Store as `point[type] = Math.round(total)`
3. Return `{ data, types }`

**Props:**
```typescript
interface InvestmentChartProps {
  projections: InvestmentProjection[];
  monthLabels: string[];   // from patrimonyData starting at currentMonthIndex
  globalUsdRate: number;
```
